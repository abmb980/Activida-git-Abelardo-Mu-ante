# Actividad Git – Entornos de Desarrollo

**Alumno:** Abelardo Muñante  
**Módulo:** Entornos de Desarrollo  

## Descripción del proyecto
Este proyecto es una práctica para aprender el uso básico de Git, incluyendo:
- Creación de repositorios
- Uso de commits
- Trabajo con ramas
- Fusión de ramas
- Subida del proyecto a GitHub

El proyecto consiste en una página web sencilla desarrollada con HTML y CSS.
